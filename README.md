# NLP-Tutorial

Hands-on NLP practice notebooks and datasets.

## Project Structure

This repository is organized into two learning tracks:

1. `basic_and_advanced_preprocessing`
	- Covers text cleaning and preprocessing workflows.
	- Includes:
	  - `basic_text_preprocessing.ipynb`
	  - `basic_and_advanced_text_cleaning.ipynb`
	  - `IMDB_Dataset.csv`
	  - `tweet_emotions.csv`
	  - `test_data_cleaning.csv`

2. `word2vec`
	- Covers training and exploring a Word2Vec model.
	- Includes:
	  - `word2vec.ipynb`
	  - `Book1.txt` to `Book7.txt`
	  - `Harry_Potter_all_books_preprocessed.txt`

## How To Run

1. Clone the repository.
2. Open the notebooks in Jupyter or VS Code.
3. Install dependencies used in notebooks, such as:
	- `nltk`
	- `gensim`
	- `scikit-learn`
	- `plotly`

## Notes

- Large dataset files are included for learning convenience.
- If any notebook requires additional NLTK resources, download them in a setup cell.
